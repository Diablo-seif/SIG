Features
--------

* Select icons for any record in Odoo by using this widget.

* Harness the power of Font Awesome in your app.

* Font Awesome v4.7.x Compatibility.

* Uses `FontAwesome Bootstrap Icon Picker <https://farbelous.io/fontawesome-iconpicker/>`_.

.. class:: text-left

Usage
-----

#. Define the field as *Char*.

#. Use the following attribute for your field in the view::

   <field name="<name>" widget="icon_selector"/>
