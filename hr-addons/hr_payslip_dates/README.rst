Hr Payslips Default Dates
=========================

Hr Payslips Default Dates

Features
--------

#. Payslip Default Date From And Date To Depend On System Parameters.
#. Change Name Payslip Depend On System Parameters.
#. The System Parameters Configuration From Module `hr compute attendance`
   but not must be installed now (optional configuration).
#. You Can Define the Parameters using the keys:
   1. `day_start_month` number of start day of month
   2. 'start_month' have 2 value (current/previous)
