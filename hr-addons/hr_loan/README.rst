Loan Management
===============

.. class:: text-left

Features
--------

* Configuration for loans
  1. Loan Type (percentage from employee salary or amount)
  2. Loan Max (maximum amount for loan request)
  3. Loan Max Months (maximum number of months for loan request)
  4. Allow Multiple loans(allow multiple loan request)
