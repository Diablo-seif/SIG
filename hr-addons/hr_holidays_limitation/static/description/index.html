HR Time Off Limitations

Overview

Provide the ability to restrict employees to submit time off request before specific period.

Features

  • Set number of days limitation option for time off type with Day option to prevent request before configured days.
  • Set number of hours limitation option for time off type with Hour option to prevent request before configured hours
  • Set number of days to allow employees to request time off before contract start date (Optional based on each contract)
  • Allow time off Administrator can request without any limitation

Configuration

Update contract to check for "Time Off Limitation"

From time off type: Set "Contract Limitation" days. Example limit employee request after 90 from contract start date

From time off type: Set "No Of Days" needed to request time off before start date. Example limit employee request before 3 days from take time off

Notes

  • If you choose time off type by hours, you need to set number of hour instead of days. Example: employee must request excuse before 2 hours

How to Work

Employee request time off before contract limitation

Employee request time off before number of days limitation

Employee request time off before number of hours limitation

Notes:This module is tested and works on Odoo Community and Enterprise Edition.
