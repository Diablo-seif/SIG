Time Off Limitations
=======================

.. class:: text-left

Features
--------

- No. of days limitation option for time off type with Day option to prevent request before configured days
- No. of hours limitation option for time off type with Half-day and Hour option to prevent request before configured hours
- Only time off manager can request without any limitation
