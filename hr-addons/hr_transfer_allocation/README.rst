Transfer/Reset Allocations
==========================

.. class:: text-left

Features
--------

- Ability to Reset Allocations of specific types
- Ability to Transfer Allocations of specific types to specific type

.. class:: text-left

How to Use
----------

#. Under **Employees**
#. Select some employees
#. **Actions** > **Transfer/Reset Allocations**
#. select leaves types
#. select Transfer or Reset
