HR Organization levelling
=========================

.. class:: text-left

Features
--------

- Set maximum organization levelling at general settings configuration
- Ensure that organization departments wont exceed the configured level


.. class:: text-left

How to Use
----------

#. Click on **Settings** > **General Settings**
#. At **Companies** Section Configure **Organization levelling** and click **SAVE**
#. If the levels of departments exceeded the configured level it will raise error
