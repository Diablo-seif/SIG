Manpower Plan
=============

.. class:: text-left

Features
--------

- **Manpower Plan** module to plan the organization manpower
- Share the plan with organization department manager to set their requirements
- Each department manager can set the required positions
- HR can set the budget for each department
- Overall plan budget will automatically computed based on department budget


.. class:: text-left

How to Use
----------


#. **Manpower Plan** menu under **Recruitment** menu
#. Set plan period date and the responsible for the plan
#. Select the departments that the plan will cover
#. **SHARE** button will schedule an activity for all department managers to set their requirements
#. When every department set their requirements the responsible will set the budget for departments them submit to the Chairman
#. Chairman will be able to approve or reject the budget
#. After that responsible will submit overall plan to the chairman to approve or reject
