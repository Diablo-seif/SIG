Job Description
===============

.. class:: text-left

Features
--------

- job description template
- HTML job description

.. class:: text-left

How to Use
----------

#. Click **Recruitment** icon > **Configurations** > **Job Description Templates**
#. Create New Template to be used in job position
#. Click on **Job Position** menu and click edit
#. Choose the created template its description will displayed in job description
