Action Reason
=============

.. class:: text-left

Features
--------

- Define action reasons for reject or any other actions that needs reason

.. class:: text-left

How To Use
----------

#. Define `ir.actions.act_window` with `<field name="context">{'default_res_model': 'model.model'}</field>`
#. Define `menu` for the `action` for your model to define reasons that will be used
