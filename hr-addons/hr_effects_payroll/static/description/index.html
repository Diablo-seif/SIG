HR Payslip Effect

Overview

Provide the ability to record any addition or deduction amount for employees.

It can be used to compensate employees for the per diem, Bonus, Overtime, late attend etc...

Features

  • Define additions types.
  • Define deductions types.
  • Control users view only additions and deductions.
  • Control users full access on additions and deductions types.
  • Control users full access on additions and deductions.
  • Record additions and deduction amount in any time.
