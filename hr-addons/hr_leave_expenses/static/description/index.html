HR Time Off Expense

Overview

Provide the ability to integrate Employee Expense with Time Off. Many businesses use cases of integration possibility between time off and Employee Expenses.

It can be used to compensate employees for the per diem, including the employee spending some of the expenses related to the company, which can be far from the workplace as an example accommodation, meal and housing allowance.

Features

  • Create template for expense.
  • Customize per diem for specific employees, employee tags, departments and job positions.
  • Set per diem in other currency.
  • Use Odoo Time off and expense cycle.

Configuration

Create new expense product

Create new time off type with day and mark on “Link To Expenses”

Click on “Add a line” and specify product, amount per day then which employees will get this product

Notes

  • If you do not choose any criteria Odoo will apply it to all employees
  • You can specify multi criteria as example employee with tag “Consultant” or in department “Sales”, etc.…

How to Work

Employee request time off with type “Travel” with an example 3 days.

Based on approval cycle, when request is approved, Odoo will create new expense sheet and open it automatically (based on criteria defined in time off type) to reimburse employee.

Notes: Expense sheet will be created in submitted stage to allow to approve and paid by accountant.

Notes:This module is tested and works on Odoo Community and Enterprise Edition.
