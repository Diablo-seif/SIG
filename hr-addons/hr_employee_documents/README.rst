HR Employee Document
====================

.. class:: text-left

Features
--------

- New menu **Employee Documents** under **configuration** menu to define required employee documents
- **Employee Documents** will be listed by default when creating new employee
- If document has expiration, **expiration date** field will be required if the document selected for employee.
- If document has expiration date an activity will be created month before the expiration date.
