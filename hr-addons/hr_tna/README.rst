Training Needs Assessment
=========================

.. class:: text-left

Features
--------

- **TNA** module to plan organization training needs assessment
- Share the TNA with organization department manager to set their requirements
- Each department manager can set the required courses
- HR can set the budget for each department
- Overall TNA budget will automatically computed based on department budget


.. class:: text-left

How to Use
----------


#. **Training Needs Assessment** menu under **Employee** menu
#. Set TNA period date and the responsible for the plan
#. Select the departments that the TNA will cover
#. **SHARE** button will schedule an activity for all department managers to set their requirements
#. When every department set their requirements the responsible will set the budget for departments then submit to the Chairman
#. Chairman will be able to approve or reject the budget
#. After that responsible will submit overall plan to the chairman to approve or reject

#.
